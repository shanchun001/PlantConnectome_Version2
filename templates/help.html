Help

Search Types

  • Gene/Word Search: Searches for entities containing the search term as a word. Best for general searches. For example, searching "CESA" will match "CESA1", "CESA3", and "CESA complex".
  • Substring Search: Finds entities containing the search term as a substring (excluding exact matches). Useful for finding variants and partial matches.

Author Search

Search by author name using "Last Name Initials" format (e.g., "Mutwil M").

PubMed ID Search

Search by PubMed ID. Multiple IDs can be separated by semicolons.

Network Visualization

Results are displayed as an interactive network graph using Cytoscape.js. Nodes represent entities and edges represent relationships extracted from scientific literature.

Entity Types

The database contains relationships between various entity types including:

  • Genes, Proteins, and Enzymes
  • Organisms, Organs, Tissues, and Cell Types
  • Metabolites, Hormones, and Chemical Compounds
  • Phenotypes and Stress Responses
  • Biological Processes and Pathways
  • Treatments and Environmental Conditions
  • Genomic and Transcriptomic Features
  • Methods, Tools, and Databases
