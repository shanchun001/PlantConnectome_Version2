PlantConnectome is built from scientific literature processed with GPT assistance to extract relationships between plant science entities.

Key Features

  • Multiple Search Modes: Gene/Word and Substring searches for flexible entity discovery.
  • Interactive Network Visualization: Explore relationships using Cytoscape.js-powered network graphs.
  • Author Search: Find all relationships from papers by a specific author.
  • PubMed Integration: Search by PubMed ID to see all extracted relationships.
  • AI-Powered Summaries: Generate summaries of relationships using GPT models.
  • API Access: Programmatic access to search results via REST API.

Entity Categories

  • Plant/Organism: Plants, crops, model organisms, and plant communities
  • Gene/Protein: Genes, proteins, enzymes, transcription factors, and protein complexes
  • Chemical: Metabolites, hormones, compounds, and molecules
  • Disease/Stress: Plant diseases, abiotic stress, pathogens, and disorders
  • Cell/Compartment: Cells, organelles, tissues, and anatomical structures
  • Biological Process: Pathways, metabolic processes, signaling, and biological functions
  • Treatment: Experimental conditions, growth conditions, and interventions
  • Taxonomy: Phylum, class, order, family, genus, species, cultivar
